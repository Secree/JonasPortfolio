## Getting Started

### Prerequisites

- Node.js 18.17 or later
- npm, yarn, or pnpm

### Installation

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Customization

Edit the placeholder content in the following files:

- `src/components/Hero.tsx` - Update your name, title, and description
- `src/components/About.tsx` - Add your bio and skills
- `src/components/Projects.tsx` - Add your project details
- `src/components/Contact.tsx` - Update your contact information and social links
- `src/components/Footer.tsx` - Update footer information

## Deployment on Vercel

1. Push your code to GitHub
2. Go to [Vercel](https://vercel.com)
3. Import your repository
4. Vercel will automatically detect Next.js and deploy your site

Alternatively, use the Vercel CLI:

```bash
npm install -g vercel
vercel
```
